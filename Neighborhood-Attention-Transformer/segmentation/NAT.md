## Validation
<details>
<summary>
<b>NAT-Mini + UPerNet</b>
</summary>

Single scale:
```shell
./dist_test.sh \
    configs/nat/upernet_nat_mini_512x512_160k_ade20k.py \
    https://shi-labs.com/projects/nat/checkpoints/SEG/nat_mini_upernet.pth \
    $NUM_GPUS \
    --eval mIoU
```

Multi scale:
```shell
./dist_test.sh \
    configs/nat/upernet_nat_mini_512x512_160k_ade20k.py \
    https://shi-labs.com/projects/nat/checkpoints/SEG/nat_mini_upernet.pth \
```
